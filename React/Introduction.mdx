# Chessboard for React

This goal of the project is to give to the developer, a modern chessboard. 

## Modules

You can add or remove modules, depend on your needs.. In case you want to add arrows and warn cases, you can add an argument that adds it. The same thing is needed for timer, history, game details, enable of movement, orientation, the game mode or starting position.

## Other frameworks

If you use an another frameworks, checkout the [Github](https://github.com/playeChess/Chessboard) repository to see if it's avalaible.
